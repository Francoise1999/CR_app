# 📖 Journal Spirituel

Un journal de suivi spirituel personnel, hébergeable gratuitement sur **GitHub Pages**.

## ✨ Fonctionnalités

- ✅ Entrée quotidienne pour 10 indicateurs spirituels
- 📊 Statistiques (mensuel, annuel, global) avec graphiques
- 🎯 Objectifs journaliers et mensuels personnalisables
- 📚 Gestion de bibliothèque de livres chrétiens
- 📄 Génération de rapports journaliers, mensuels et annuels
- ⬇ Export CSV (compatible Excel) et PDF
- 📱 Interface responsive (mobile, tablette, ordinateur)
- 💾 Données stockées localement (localStorage — aucun serveur requis)

## 🚀 Déploiement sur GitHub Pages

### Étape 1 : Créer un repository GitHub
1. Connectez-vous sur [github.com](https://github.com)
2. Cliquez sur **"New repository"**
3. Nommez-le `journal-spirituel` (ou ce que vous voulez)
4. Cochez **"Public"** (requis pour GitHub Pages gratuit)
5. Cliquez **"Create repository"**

### Étape 2 : Uploader les fichiers
1. Dans votre repository, cliquez **"Add file" → "Upload files"**
2. Glissez-déposez les 3 fichiers : `index.html`, `style.css`, `app.js`
3. Cliquez **"Commit changes"**

### Étape 3 : Activer GitHub Pages
1. Allez dans **Settings** → **Pages**
2. Sous "Source", sélectionnez **"Deploy from a branch"**
3. Choisissez la branche **"main"** et le dossier **"/ (root)"**
4. Cliquez **Save**

### Étape 4 : Accéder à votre site
Votre site sera disponible à :
```
https://[votre-username].github.io/journal-spirituel/
```

> ⚠️ **Note :** Les données sont stockées dans le `localStorage` de votre navigateur. 
> Elles sont **locales à chaque appareil**. Pour synchroniser entre appareils, 
> utilisez la fonction **Export CSV** et réimportez sur l'autre appareil (fonctionnalité avancée).

## 📱 Utilisation sur mobile

Ajoutez le site à votre écran d'accueil :
- **iPhone** : Safari → Partager → "Sur l'écran d'accueil"
- **Android** : Chrome → Menu → "Ajouter à l'écran d'accueil"

## 🔒 Confidentialité

Toutes vos données restent **sur votre appareil**. Rien n'est envoyé sur internet.
